# Copyslab

A snippet manager for Linux. Click a snippet to copy it to the clipboard — no confirmation dialog. Tag hierarchy (`tag/subtag`), full-text search across names and content.

Data is stored as plain text files on disk. Sync via Dropbox or Git is up to you. Built with Tauri v2 and a Rust backend — not Electron.
